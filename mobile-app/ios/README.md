# MLJ NET iOS Platform

This directory contains the iOS native project for the MLJ NET mobile app,
managed by [Capacitor](https://capacitorjs.com/).

## Directory Structure

```
ios/
├── App/
│   ├── App/
│   │   ├── AppDelegate.swift     # App delegate (generated by Capacitor)
│   │   ├── Info.plist            # App configuration & permissions
│   │   └── LaunchScreen.storyboard  # Launch screen
│   ├── Assets.xcassets/          # App icons and images
│   └── Base.lproj/               # Storyboards
├── Podfile                       # CocoaPods dependencies
└── App.xcworkspace/              # Xcode workspace
```

## Custom Files

The following files are custom overrides that will NOT be overwritten by Capacitor sync:

- `App/App/Info.plist` - App permissions, ATS config, orientation
- `App/App/AppDelegate.swift` - App lifecycle and deep link handling

## ⚠️ Important

- iOS builds **MUST** be done on macOS with Xcode 15+ installed.
- Do NOT modify files generated by Capacitor (`cap sync`) unless necessary.
- Always run `npx cap sync ios` after modifying `capacitor.config.ts`.
- App icons should be added to `App/Assets.xcassets/AppIcon.appiconset/`.
