# MLJ NET Android Platform

This directory contains the Android native project for the MLJ NET mobile app,
managed by [Capacitor](https://capacitorjs.com/).

## Directory Structure

```
android/
├── app/
│   └── src/
│       └── main/
│           ├── AndroidManifest.xml       # App permissions & configuration
│           ├── java/                      # Java/Kotlin source code (generated by Capacitor)
│           └── res/
│               ├── values/
│               │   ├── colors.xml         # Dark theme color palette
│               │   ├── strings.xml        # App strings & messages
│               │   └── styles.xml         # Dark theme styles
│               ├── xml/
│               │   ├── network_security_config.xml  # Network security
│               │   └── file_paths.xml     # FileProvider paths
│               └── mipmap-*/             # App icons (add your own)
├── build.gradle                          # Root build config
├── variables.gradle                       # SDK versions & config
└── settings.gradle                        # Project settings
```

## Custom Files

The following files are custom overrides that will NOT be overwritten by Capacitor sync:

- `app/src/main/AndroidManifest.xml` - Permissions and activity config
- `app/src/main/res/values/colors.xml` - Dark theme colors
- `app/src/main/res/values/styles.xml` - Dark status bar and splash
- `app/src/main/res/values/strings.xml` - App name and strings
- `variables.gradle` - SDK and build version config

## ⚠️ Important

Do NOT modify files in `app/src/main/java/` as they are generated by Capacitor.
To customize the app, edit the files listed above or `capacitor.config.ts`.
